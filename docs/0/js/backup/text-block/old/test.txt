<ruby>山田太郎<rt>やまだたろう</rt></ruby>はどう出るか確認してみる。
<ruby>山田<rt>やまだ</rt>太郎<rt>たろう</rt></ruby>はどう出るか確認してみる。
<ruby>山<rt>やま</rt>田<rt>だ</rt>太<rt>た</rt>郎<rt>ろう</rt></ruby>はどう出るか確認してみる。
<ruby>山<rt>やま</rt></ruby><ruby>田<rt>だ</rt></ruby><ruby>太<rt>た</rt></ruby><ruby>郎<rt>ろう</rt></ruby>はどう出るか確認してみる。
以上のことから終了タグまではまとめて全部一度に表示されると判明した。
rubyタグは一字ごとにタグを作ることで一字ずつ表示できる。かなり面倒くさい。

TextBlock（テキストブロック）について。

テキストをブロックに分けます。
改行コード１つの直後に続くテキストは同一ブロック内です。

改行コード２つ以上の直後に続くテキストは次の別ブロックです。



これは４ブロック目です。
ブロック間にたくさん改行があっても空ブロックができたりはしません。

ブロック内で改行表示したいときは改行エスケープします１。\n\nこんなふうにね。

ブロック内で改行表示したいときは改行エスケープします２。
\nこんなふうにね。

改行エスケープだけの行は特例としてエスケープした数だけ表示します。
\n
こんなふうにね。
これにより台本と出力結果で見た目の行数が一致してわかりやすくなります。

改行エスケープは連続できます。
\n\n\n
こんなふうにね。

改行エスケープ行は連続しても有効です。
\n
\n\n
\n
ほらね。

改行エスケープのエスケープはバックスラッシュをひとつ多くつける。
\\nのようにね。\\nのように途中や末尾でも可\\n
